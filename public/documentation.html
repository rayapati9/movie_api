Users can sign up and create list of their favourite movies.

Business Logic URL Endpoint HTTP Method Request Body Data Format Response Body Data Format
Get a list of all movies /movies GET None JSON object holding data about all movies
Get information about one specific movie by title, use %20 for spaces /movies/:title GET None JSON object about a specific movie
Get data about genres by title/name movies/Genres/:genre Get None JSON
Get data about director by name movies/Directors/:name GET None JSON
Allow new users to register /users POST JSON JSON
Allow users to update their user information /users/:username PUT JSON JSON
Allow users to add a movie to their list of favourites /users/:username/movies/movieID POST JSON JSON
Allow users to remove a movie from their list of favourites /users/:username/movies/movieID DELETE None Message
Allow existing users to deregistor /users/:username DELETE None Message
